# Manual Admin (V11)

- /admin → painel admin
- Whitelist: limite 5 ativos (o banco bloqueia o 6º)
- Categorias: crie por tipo (cursos/livros/arquivos)
- Itens: crie curso/livro/arquivo dentro da categoria
  - Tags: separar por vírgula para filtros
- Cursos: crie módulos/aulas; ajuste ordem com ↑ ↓
- Notificações: publicar/despublicar
- Início: editar home
- Tema/Layout: ajuste tokens (accent/bg/card/text/muted/radius)
- Importador Telegram: cole JSON e ele cria estrutura
- Suporte: inbox e respostas
